## Installation

```bash
$ pnpm install
```

## Running the app

### Using Docker Compose
 
Start the container:
   ```bash
   docker-compose up
   ```
   
### Without Docker

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

Access the app at `http://localhost:3010`.
